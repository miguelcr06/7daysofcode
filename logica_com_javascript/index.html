Dia 1

Resolver problema com tipos de váriaveis quando comparar valor

Para este desafio, nos foi dado as seguintes variáveis:

let numeroUm = 1

let stringUm = '1'

let numeroTrinta = 30

let stringTrinta = '30'

let numeroDez = 10

let stringDez = '10'

Nossa tarefa é comparar os valores das variáveis e identificar quais são iguais e quais são diferentes, considerando seus tipos.

Caso queira, você pode clicar no botão ao lado para ver a instrução completa


Solução

Para resolver este desafio, precisamos comparar as variáveis fornecidas considerando tanto o valor quanto o tipo de dado.

Em JavaScript, utilizamos dois operadores principais para comparações:

  • == (igualdade solta): Compara apenas o valor, ignorando o tipo.
  • === (igualdade estrita): Compara valor e tipo simultaneamente.

Analisando as variáveis:

  • numeroUm = 1 (número)
  • stringUm = '1' (string)
  • numeroTrinta = 30 (número)
  • stringTrinta = '30' (string)
  • numeroDez = 10 (número)
  • stringDez = '10' (string)

Comparações de igualdade solta (==):

  • numeroUm == stringUm → true (valor igual, tipos diferentes)
  • numeroTrinta == stringTrinta → true
  • numeroDez == stringDez → true

Comparações de igualdade estrita (===):

  • numeroUm === stringUm → false (tipos diferentes)
  • numeroTrinta === stringTrinta → false
  • numeroDez === stringDez → false

Para identificar se são iguais ou diferentes, use === para verificar ambos os aspectos.

Dia 2

Capturar e armazenar valores em variáveis

Para este desafio, você deve criar um programa que simule um cadastro personalizado, como em um site.

O programa deve perguntar ao usuário 3 perguntas:

  • Qual o seu nome?
  • Quantos anos você tem?
  • Qual linguagem de programação você está estudando?

No final, exibir a mensagem: "Olá [nome], você tem [idade] anos e já está aprendendo [linguagem]!"

Exercício opcional: Após exibir a mensagem, perguntar "Você gosta de estudar [linguagem]?" com botões SIM e NÃO, e mostrar uma resposta baseada na escolha.

Caso queira, você pode clicar no botão ao lado para ver a instrução completa


Solução

Para resolver este desafio, usamos variáveis para capturar e armazenar as respostas do usuário.

Em JavaScript, podemos usar prompt() para obter entrada do usuário:

let nome = prompt("Qual o seu nome?");
let idade = prompt("Quantos anos você tem?");
let linguagem = prompt("Qual linguagem de programação você está estudando?");

alert(`Olá ${nome}, você tem ${idade} anos e já está aprendendo ${linguagem}!`);

As variáveis nome, idade e linguagem armazenam as respostas.

Para o exercício opcional, usamos confirm() para uma pergunta sim/não:

let gosta = confirm(`Você gosta de estudar ${linguagem}?`);
if (gosta) {
    alert("Muito bom! Continue estudando e você terá muito sucesso.");
} else {
    alert("Ahh que pena... Já tentou aprender outras linguagens?");
}

Isso torna a experiência mais interativa e personalizada.

Dia 3

Decisões e Loops

Para este desafio, você deve criar um jogo interativo onde o usuário faz escolhas sobre seu caminho na programação.

O programa deve permitir ao usuário escolher entre Front-End ou Back-End, depois tecnologias específicas, especialização ou Fullstack, e finalmente listar tecnologias que deseja aprender.

Use estruturas de controle como if/else e loops para guiar o fluxo baseado nas escolhas.

Caso queira, você pode clicar no botão ao lado para ver a instrução completa


Solução

Para resolver este desafio, usamos estruturas de controle de fluxo como if/else para ramificações e loops while para coletar múltiplas tecnologias.

Exemplo de código:

let area = prompt("Você quer seguir para área de Front-End ou Back-End?");
if (area === "Front-End") {
    let tech = prompt("Quer aprender React ou Vue?");
    // ...
} else if (area === "Back-End") {
    let tech = prompt("Quer aprender C# ou Java?");
    // ...
}

let especializar = prompt("Quer se especializar ou se tornar Fullstack?");
if (especializar === "especializar") {
    alert("Especialize-se na sua área!");
} else {
    alert("Torne-se Fullstack!");
}

let tecnologias = [];
let continuar = "sim";
while (continuar === "sim") {
    let tech = prompt("Qual tecnologia gostaria de aprender?");
    tecnologias.push(tech);
    continuar = prompt("Tem mais alguma? (sim/não)");
}
alert("Tecnologias: " + tecnologias.join(", "));

Isso cria um fluxo interativo baseado nas escolhas do usuário.

    Interação: Exemplo com Número Fixo (7)

    Teste o exemplo com número fixo 7. Você tem 3 tentativas.

    Solução

    Primeiro, exemplo com número fixo (7):



    Agora, versão com número aleatório:

    let numeroSecreto = Math.floor(Math.random() * 11);
let tentativas = 3;
let acertou = false;

while (tentativas > 0 && !acertou) {
    let chute = parseInt(prompt("Adivinhe o número (0-10):"));
    if (chute === numeroSecreto) {
        alert("Parabéns! Você acertou!");

        } else {
            alert(`Você perdeu! O número era ${numeroSecreto}.`);
        }
    }
}

    Dia 6

    Lista de Compras com Remoção

    Para este desafio, você melhora o programa do Dia 5 adicionando a funcionalidade de remover itens!

    O programa perguntará se você quer adicionar ou remover uma comida da lista.

    Se escolher remover, exibe os itens disponíveis e você escolhe qual remover.

    Se o item não existir, mostra uma mensagem informando isso.

    Use métodos de array como indexOf() e splice() para remover itens.

    Caso queira, você pode clicar no botão ao lado para ver a instrução completa


    Solução

    Exemplo com adição e remoção:



    Agora, versão com entrada do usuário:

    let shoppingList = { /* ... */ };

function adicionarItem(nome, categoria) {
    shoppingList[categoria].push(nome);
}

function removerItem(nome) {
    for (let categoria in shoppingList) {
        let index = shoppingList[categoria].indexOf(nome);
        if (index > -1) {
            shoppingList[categoria].splice(index, 1);
            return true;
        }
    }
    return false;
}

while (true) {
    let acao = prompt("Adicionar ou remover?");
    if (acao === "adicionar") {
        let item = prompt("Qual item?");
        let cat = prompt("Qual categoria?");
        adicionarItem(item, cat);
    } else if (acao === "remover") {
        let item = prompt("Qual item remover?");
        if (!removerItem(item)) {
            alert("Item não encontrado!");
        }
    }
}

    Dia 7

    Calculadora com Funções

    Para este desafio, você deve criar uma calculadora com operações em funções separadas.

    O programa exibe um menu com opções: Soma, Subtração, Multiplicação, Divisão, Sair.

    Usuário escolhe uma operação, insere dois números, e vê o resultado.

    Cada operação deve ser uma função diferente no seu código.

    Quando sair, mostra "Até a próxima".

    Caso queira, você pode clicar no botão ao lado para ver a instrução completa


    Solução

    Exemplo com as 4 operações básicas:



    Estrutura do código:

    // Funções de operação
function somar(a, b) { return a + b; }
function subtrair(a, b) { return a - b; }
function multiplicar(a, b) { return a * b; }
function dividir(a, b) { 
    if (b === 0) return "Erro: Divisão por zero!";
    return a / b; 
}

// Menu principal
function mostrarMenu() {
    let operacao = prompt("Escolha: +, -, ×, ÷ ou Sair");
    
    if (operacao === "Sair") {
        alert("Até a próxima!");
        return;
    }
    
    let num1 = parseFloat(prompt("Primeiro número:"));
    let num2 = parseFloat(prompt("Segundo número:"));
    
    let resultado;
    if (operacao === "+") resultado = somar(num1, num2);
    else if (operacao === "-") resultado = subtrair(num1, num2);
    else if (operacao === "×") resultado = multiplicar(num1, num2);
    else if (operacao === "÷") resultado = dividir(num1, num2);
    
    alert(resultado);
    mostrarMenu(); // Volta ao menu
}
