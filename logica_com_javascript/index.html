Dia 1

Resolver problema com tipos de váriaveis quando comparar valor

Para este desafio, nos foi dado as seguintes variáveis:

let numeroUm = 1

let stringUm = '1'

let numeroTrinta = 30

let stringTrinta = '30'

let numeroDez = 10

let stringDez = '10'

Nossa tarefa é comparar os valores das variáveis e identificar quais são iguais e quais são diferentes, considerando seus tipos.

Caso queira, você pode clicar no botão ao lado para ver a instrução completa

Solução

Para resolver este desafio, precisamos comparar as variáveis fornecidas considerando tanto o valor quanto o tipo de dado.

Em JavaScript, utilizamos dois operadores principais para comparações:

  • == (igualdade solta): Compara apenas o valor, ignorando o tipo.
  • === (igualdade estrita): Compara valor e tipo simultaneamente.

Analisando as variáveis:

  • numeroUm = 1 (número)
  • stringUm = '1' (string)
  • numeroTrinta = 30 (número)
  • stringTrinta = '30' (string)
  • numeroDez = 10 (número)
  • stringDez = '10' (string)

Comparações de igualdade solta (==):

  • numeroUm == stringUm → true (valor igual, tipos diferentes)
  • numeroTrinta == stringTrinta → true
  • numeroDez == stringDez → true

Comparações de igualdade estrita (===):

  • numeroUm === stringUm → false (tipos diferentes)
  • numeroTrinta === stringTrinta → false
  • numeroDez === stringDez → false

Para identificar se são iguais ou diferentes, use === para verificar ambos os aspectos.
